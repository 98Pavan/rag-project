# RAG System from Scratch 🧠

A complete Retrieval Augmented Generation (RAG) system built from scratch using Python and Google Gemini API.

## What is RAG?
RAG combines document retrieval with AI generation to answer questions from your own documents!

## How it Works
## Steps Built
- ✅ Step 1: Document Loading
- ✅ Step 2: Text Chunking with Overlap
- ✅ Step 3: Embeddings using Gemini (3072 dimensions)
- ✅ Step 4: Cosine Similarity Search
- ✅ Step 5: Answer Generation using Gemini

## Technologies Used
- Python 3.13
- Google Gemini API
- gemini-embedding-001 (Embeddings)
- gemini-2.5-flash (Generation)

## How to Run
1. Install dependencies:
   pip install google-genai
2. Add your Gemini API key in rag.py
3. Run the project:
   python rag.py
